1. 로프 리스트를 낮은 중량순(오름차순)으로 정렬시킨다.
	- for문으로 비교할 때, 낮은 중량순으로 비교하면, 인덱스 0번에는 제일 낮은 로프부터 제일 높은 로프까지 계산하고
	  인덱스 1번에는 제일 낮은 로프를 제외한 제일 낮은 로프부터 제일 높은 로프까지 계산하므로
	  각각의 로프들을 전부 사용하는 경우의 수를 계산하기 편리하기 때문에 오름차순으로 정렬한다.
2. for문을 통해 로프의 인덱스에 접근하여 계산
	- 최댓값을 구하는 문제이므로 int max 변수를 만들어 if문을 통해 비교
		- cur_max = rope[i] * (n - i)
			- n - i를 하게 되면 현재 인덱스를 비교 후, 이미 비교한 로프를 제외시키고 비교하기 위해 
			  (전체 - 비교했던 로프 수)를 계산한다.
			- 그 수를 rope의 중량에 곱해야 최대 하중을 구할 수 있다.
				- 모든 로프가 버티는 최대하중 / 비교하는 로프 수 = 각 로프당 걸리는 최대 하중
				  모든 로프 최대 하중 = 각 로프 최대 하중 * 비교하는 로프 수
		- if cur_max > max ? max = cur_max : return max
3. return max
